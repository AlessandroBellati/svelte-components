# svelte-components
Set of Svelte components.

## Installation
```bash
npm install https://github.com/AlessandroBellati/svelte-components
```

## CSS import
[+layout.svelte](/src/routes/+layout.svelte)
```svelte
<script>
    import "/node_modules/svelte-components/dist/css/index.css"
</script>

<slot></slot>
```
